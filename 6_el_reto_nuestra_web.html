6: El Reto: Nuestra Web

Descripción del Producto Final

Vuestro reto final es construir una página web en Google Sites que funcione como una oficina de ventas real. La web debe ser visual, profesional y fácil de navegar para un cliente que busca servicios de jardinería o floristería.

Secciones Obligatorias de la Web
Para superar el reto, vuestro sitio debe tener al menos estas 4 páginas:

1. Página de Inicio (Home):

Un eslogan potente (creado con ayuda de IA).
Una foto impactante de vuestra empresa o equipo.
¿Quiénes somos? Una breve historia que transmita confianza.

2. Catálogo de Productos y Servicios:

Floristería: Ramos, eventos, decoración de iglesias, etc.
Mantenimiento: Poda en altura, riego automático, diseño de jardines.
Nota: Cada servicio debe llevar una descripción técnica precisa y una foto de calidad.

3. Galería de Proyectos:

Imágenes de trabajos realizados en el taller del instituto (vuestros ramos, vuestros trasplantes).
Antes y después de un mantenimiento ficticio.

4. Atención al Cliente y Privacidad:

Formulario de contacto.
Mapa de ubicación (podéis usar el del instituto).
Aviso Legal: Un pequeño texto donde indiquéis que los datos del cliente están seguros con vosotros.

Recomendaciones: 
Paso 1: Elegid una de las plantillas de Google Sites (recomendamos "Cartera" o "Evento").
Paso 2: Insertad los textos que corregimos en Google Docs.
Paso 3: Subid vuestras fotos de Canva. ¡Cuidado con la resolución!
Paso 4: Revisad que todos los botones funcionen antes de "Publicar".
